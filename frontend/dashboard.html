Complete Setup Guide
Part 1 — Backend Deployment
1

Clone & Install

Download the project. Navigate to backend/ and run: npm install

2

Configure Environment

Copy .env.example to .env. Set JWT_SECRET to a long random string. Set your admin username and password.

3

Deploy to Railway / Render / VPS

Railway (railway.app): Connect GitHub repo → Deploy. Set environment variables in the Railway dashboard. Get your public URL.

4

Host the Dashboard

Open frontend/dashboard.html → Set the API_URL variable at the top to your backend URL. Deploy to Netlify, Vercel, or GitHub Pages.

Part 2 — Tampermonkey Setup
5

Install Tampermonkey

Install the Tampermonkey browser extension from the Chrome Web Store or Firefox Add-ons. Use Chrome for best compatibility.

6

Add Your Instagram Account

In the dashboard, go to Accounts → add your Instagram account. To get the sessionid: log into Instagram in Chrome → F12 → Application → Cookies → instagram.com → find sessionid → copy value.

7

Create a Campaign

Go to New Campaign → fill in all fields → click Create Campaign. Note the campaign ID from the Campaigns tab.

8

Generate & Install Script

Go to Script Config → select your campaign → enter your backend URL → copy the generated script. In Tampermonkey: click icon → Create New Script → paste → Save (Ctrl+S).

9

Start the Bot

Open https://www.instagram.com in the browser with Tampermonkey installed. The script runs automatically. Watch Live Logs to confirm it's working. Keep this browser window open.

Part 3 — UptimeRobot (24/7 Keepalive)
10

Create Free Account

Sign up at uptimerobot.com (free plan gives you 50 monitors with 5-min intervals).

11

Add HTTP Monitor

Click Add New Monitor → Type: HTTP(S) → Friendly Name: InstaReach → URL: your backend /ping endpoint → Monitoring Interval: 5 minutes → Create Monitor.

12

Set Up Alerts

Add email or Telegram notifications so you're alerted if the backend goes down. Telegram alerts are instant and free.

13

Keep Browser Running 24/7

UptimeRobot keeps your backend alive. For the browser (Tampermonkey) to run 24/7, leave the PC/Mac running with the Instagram tab open. Disable screen sleep. Alternatively use a cheap VPS with a real browser via Xvfb + Chrome.

⚠ Safety Limits: Max 150 DMs/day per account · Always use random delays (8-15s) · Warm up new accounts gradually (30/day for first week) · Never use the exact same message across all accounts · Do not DM verified accounts or accounts with 100k+ followers — Instagram flags this pattern.
